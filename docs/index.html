Features

🔍

Deep Research

Multi-step web search, cross-source contradiction detection, structured report synthesis.

🧪

Eval-Driven

Built-in evaluation suite with 3+ research tasks and graded quality checks. Run with --eval.

🔄

Self-Improving

GEPA loop evolves prompt strategies across generations. Discovers what works best automatically.

📝

Self-Patching

Generates _self_patch.py with learned improvements. Apply with one command.

🤖

Local LLM Grading

Auto-detects Ollama models for quality scoring. Falls back to heuristic grading when no GPU available.

📦

Single File

Zero infrastructure. One Python file. pip install from GitHub and you’re running.

Quick Start

# Install
pip install git+https://github.com/bigknoxy/research-agent.git

# Run a research query
python research_agent.py "What are the latest developments in RAG?"

# Run the eval suite
python research_agent.py --eval --eval-runs 1

# Run self-improvement loop
python research_agent.py --gepa --gepa-generations 3

# Full dogfood pipeline (eval + GEPA + self-patch)
python research_agent.py --dogfood

# Apply learned improvements
python _self_patch.py

# Update to latest
pip install --upgrade git+https://github.com/bigknoxy/research-agent.git

# Uninstall
pip uninstall research-agent

How It Works

research-agent is built around a pipeline of specialized agents:

Query → TaskPlanner → WebCrawler → ContradictionEngine → ReportSynthesizer → Report
         ↑              ↑            ↑                     ↑
         └─── Prompt strategies evolved by GEPA loop ──────┘

The GEPA (Generative Evolutionary Prompt Adaptation) loop evaluates prompt strategy variants against a test suite, selects winners by accuracy, and mutates them to discover better strategies. All improvements self-patch into the script.

Why?

Most AI research tools are closed-source, require API keys, or depend on heavy infrastructure. research-agent is:

  • Free — no API keys, no subscription
  • Private — runs entirely on your machine
  • Self-improving — gets better with every run via GEPA
  • Auditable — single file, readable code, deterministic evals
